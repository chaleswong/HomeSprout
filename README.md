# 🌱 HomeSprout — Skye 的成长档案系统

HomeSprout 是一款专为儿童（Skye）设计的 **Local-First 家庭数字成长档案与多维兴趣探索系统**。系统采用现代 Web 技术，配合本地轻量级大模型（Ollama），实现了对儿童日常生活创作（绘画、积木、编程、舞台表现等）的多媒体投喂、智能分类、多维兴趣画像以及大屏影院级的交互体验。

---

## ✨ 核心特性

### 1. 🕐 时光轴与成长印记 (Timeline)
- **多媒体流展示**：以时间为轴线，采用精致的磨砂玻璃（Glassmorphism）卡片卡流式渲染 Skye 的每一次投喂。
- **高光时刻收藏**：支持一键标记“高光作品”，在个人作品集中优先呈现。
- **智能统计摘要**：实时显示“全部作品”、“高光时刻”以及“本周新增”数量。

### 2. 🎨 智能画廊与精细分类 (Gallery)
- **Pinterest 瀑布流**：使用自适应 CSS Columns 布局，呈现零截断的高清晰度画廊。
- **无缝分类过滤**：支持“全部、积木、绘画、数独、编程、舞台表现、阅读、体育”等多分类的一键式即时筛选。
- **多图联动轮播**：支持单次上传多张关联照片（例如乐高城堡的正面与侧面），在详情页提供高保真左右滑块与下方焦点缩略图联动。

### 3. 📸 极简儿童“投喂”入口 (Upload)
- **儿童友好型 UI**：为 6 岁左右的孩子设计了三个超大触控色块按钮：“拍个照 📸”、“选视频 🎬”、“说两句 🎤”。
- **单/多图混合上传**：智能批处理，拖拽多图上传后由后台自动关联并生成封面缩略图 `thumb.webp`。
- **无感 AI 分类与简介**：后台自动检测多媒体 EXIF 元数据提取日期，并调用本地大语言模型（如 Qwen3）对上传作品的图片内容进行智能分类并自动撰写成长日记描述。

### 4. 📊 智能成长看板 (Dashboard)
- **纯 SVG 雷达图**：无需任何第三方繁重图表库，采用三角函数动态绘制高保真雷达多边形。
- **动态维度合并**：支持在后台将多个分类（如“编程”与“数独”）合并映射到相同的雷达图成长轴（如“逻辑思维能力”），并提供智能合并名称悬浮展示。
- **游戏化里程碑勋章**：根据每个分类的作品数量，自动解锁 “🌱 初级”、“🌿 中级” 到 “🌳 大师” 勋章，支持已解锁勋章的旋转光效与未解锁的模糊锁定展示。

### 5. 📺 电视大屏沙发影院 (TV Mode)
- **Gamepad 摇杆交互**：完美适配 HTML5 Gamepad API，坐沙发上即可手持蓝牙游戏手柄 / 遥控器进行 100% 键盘/鼠标无关的浏览。
- **远距离高对比度**：固定 4 列的大卡片排版，高亮选中项微浮动 `scale(1.05)` 与外发光渐变。
- **沉浸式播放**：支持音视频的自动播放与暂停控制，在播放音频时界面伴随声波动态起伏动效。

### 6. 📄 智能作品集与 PDF 编译导出 (Export)
- **CSS @page 打印规范**：优化 Paged Media 版式，实现 A4 打印预览的完美无截断分栏排版。
- **Typst + AI 智能报告**：调用本地 Typst 编译器排版引擎。导出时用户可输入 Prompt 口吻，大模型自动撰写 300 字结构化“成长总评/推荐信”，打包高光作品并输出图文并茂的 A4 PDF 报告书。

### 7. 📲 渐进式 Web 应用 (PWA)
- **可安装为桌面 App**：支持 Manifest 配置。无论在 iPad、手机还是电脑上，皆可一键“添加到主屏幕”作为独立客户端运行。
- **离线离网缓存**：Service Worker 拦截资源。静态资源采用 `Stale-While-Revalidate` 秒开缓存，API 采用 `Network-First`，离线状态下亦可正常浏览时光轴。

### 8. ⚙️ 局域网管理后台 (Admin)
- **多端主题同步**：支持暗色星空与清新浅色模式，切换主题时通过 WebSocket 广播使局域网内所有手机、电视、平板设备实现无刷新渐变过渡。
- **分类管理**：管理员可动态添加新分类、指定 Emoji 及专属渐变色，并指定其在雷达图上的映射成长轴。

## 🛠️ 技术栈

### 前端 (Client)
- **核心框架**：React 18 + Vite 6
- **路由导航**：React Router DOM v6
- **关系拓扑**：Cytoscape.js (用于展示作品间关联思维图谱)
- **样式风格**：Vanilla CSS (毛玻璃微动效、原生变量适配多主题)

### 后端 (Server)
- **核心框架**：Node.js + Express 5
- **实时同步**：WebSockets (ws)
- **文件监控**：Chokidar (自动监控 incoming 文件夹实现无感上传)
- **多媒体处理**：Sharp (智能缩放/缩略图生成)、Exifr (EXIF 元数据提取)、Node-Vibrant (提取图片主色调)
- **排版编译**：Typst CLI + Qwen3 LLM (Ollama API)

## 🚀 启动与部署

### 1. 安装依赖
在项目根目录下执行以安装所有 workspaces 依赖：
```bash
npm install
```

### 2. 本地开发模式
一键启动前端（Vite）开发服务与后端（Express）服务：
```bash
npm run dev
```
- **前端 Client** 访问地址：[http://localhost:5173](http://localhost:5173) (局域网 IP 同步开放)
- **大屏 TV 影院** 访问地址：[http://localhost:5173/tv](http://localhost:5173/tv)
- **成长看板** 访问地址：[http://localhost:5173/dashboard](http://localhost:5173/dashboard)
- **后端 Server** 地址：[http://localhost:3001](http://localhost:3001)

### 3. 生产打包与运行
构建静态资源并将前端托管到后端服务中运行：
```bash
npm run build
npm run start
```
